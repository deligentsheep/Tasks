### 关于Github基本用法

1. 打开浏览器之前下载一个网络加速软件，可以帮助我们更快地进入到Github中

2. 注册一个新用户

3. 查找仓库（一般是域名加上主人姓名）

4. 仓库管理

   - [x] 创建仓库

   - [x] 添加文件

   + [x] 提交Issue
   + [x] 学会README附链接
   
   - [x] 提交/接受 PR（请求）

#### 注意：

> + 在创建新仓库的时候一定要勾选显示README项目
> + 如果需要下载仓库压缩包，直接点击Copy里面的Download Zip即可
> + 提交issue之后可以供大家一起讨论（小型贴吧）
> + 上传pull qequest，也就是拉取请求，等待代码原主人把我的代码合并到主支中
